Launch Cockpit Connect Changelog

Connect your agent.

Choose where you want to run BlastOff, then follow the setup steps.

01

Add the BlastOff plugin marketplace.

/plugin marketplace add tinybuild/skills
02

Install the plugin in your preferred scope.

/plugin install blastoff@tinybuild-skills

Use global or user scope to prompt BlastOff from anywhere, or project scope to keep it available only in the current working folder.

03

Reload plugins so BlastOff is available.

/reload-plugins
04

Authorize the BlastOff MCP server.

/plugin

Open the plugin manager, switch to Installed with the arrow keys, find BlastOff, select BlastOff MCP, and press Enter to authorize the connection. The plugin ships with the MCP server registration built in — no separate setup.

Once you're set up, prompt:

Submit my product to DevHunt — it lives at https://yourproduct.com
01

Open your Cursor MCP config.

~/.cursor/mcp.json

Create the file if it doesn't exist.

02

Add BlastOff as an HTTP MCP server.

{
  "mcpServers": {
    "blastoff": {
      "transport": "http",
      "url": "https://blastoff.tinybuild.workers.dev/mcp"
    }
  }
}
03

Restart Cursor.

Cursor reads the MCP config on launch. After restart, BlastOff's 6 tools become available to the agent.

Then prompt in Composer:

Use BlastOff to list all directories I can submit to
01

Open the Codex MCP config.

~/.codex/config.toml
02

Add BlastOff as an MCP server.

[mcp_servers.blastoff]
transport = "http"
url = "https://blastoff.tinybuild.workers.dev/mcp"
03

Restart Codex.

Codex picks up the MCP server on next launch. The agent will see BlastOff's tools in its tool list.

Then prompt:

Get my BlastOff profile for https://yourproduct.com
01

Point your MCP-compatible agent at the BlastOff endpoint.

https://blastoff.tinybuild.workers.dev/mcp

Transport: HTTP. Protocol: JSON-RPC 2.0. Authorization is optional in alpha; API key auth is coming.

02

Discover available tools.

curl -s -X POST https://blastoff.tinybuild.workers.dev/mcp \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"tools/list"}'

Returns the six BlastOff tools: get_profile, list_directories, get_recipe, generate_variants, extract_from_url, mark_submitted.

03

Or fetch the server card for auto-discovery.

https://blastoff.tinybuild.workers.dev/.well-known/mcp/server-card.json
