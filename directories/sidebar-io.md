# Sidebar.io

- **URL:** https://sidebar.io
- **Submit URL:** https://sidebar.io/submit
- **Auth:** account
- **Captcha:** unknown
- **Approval:** editorial (~few days; human-curated by Fabricio Teixeira, only 5 links/day published)
- **Free:** yes (paid tier: sponsorship slots, separate from submissions)
- **One-line:** Curated daily 5-link design newsletter running since 2012
- **Ease:** 🟡 medium
- **Ease reason:** Account-gated, strict editorial bar (no paid-only products, must be new content, no heavy downloads)

## Gotchas

- Editorial filter is real: only NEW content (a few months old max), no paid-only products, no PDFs/app-store links/plugin installs.
- Best fit for design articles, portfolios, CodePens — not generic SaaS launches. For BlastOff users, this is for design tools or design-focused content drops, not every product.
- 5 picks per day means rejection rate is high; treat as editorial pitch.
- Daily newsletter ships to design audience; high-quality traffic if accepted.
