# Peerlist

- **URL:** https://peerlist.io
- **Submit URL:** https://peerlist.io/launchpad
- **Auth:** account (verified Peerlist profile required)
- **Captcha:** unknown (gated behind login)
- **Approval:** editorial — launches go live Monday weekly; staff picks finalized Tuesday
- **Free:** yes
- **One-line:** Weekly product launchpad for tech professionals on Peerlist.

## Gotchas

- Form is fully gated behind verified Peerlist profile — needs login to map exact selectors and char limits.
- **Project must be 100% complete on the Work tab** before launching, or the launch fails.
- Launch window opens Mondays weekly; can schedule for other days via project settings.
- Must launch as an individual — no company profiles, no agencies launching on behalf of clients.
- Do NOT request upvotes — explicit rule, can disqualify staff-pick eligibility.
- No plagiarized content; originality enforced.
- Cover images cap at 4; image specs not publicly documented.
- Project is created/edited on a profile Work tab, then "launched" as a separate action — autofill engine has to handle two surfaces.
